# VN Data Source Priority

## Priority 1: vnstock API

Use vnstock-backed data collection first:

- Price/history
- Financial statements
- Ratios
- Company overview
- News
- Events
- Shareholder and capital-history data where available

## Priority 2: Official Or Trusted VN Web Sources

Use later only when explicit web collection is implemented and source eligibility
is approved:

- Vietstock financial pages
- CafeF financial statement pages
- Company investor-relations pages
- Audited financial statement PDFs
- VSDC/corporate action sources

## Priority 3: Web Search

Use only for verification or supplementation after source rules are implemented.
Search results must not override collected provider data without source
traceability.
